## 1. 快速开始

### 1.1 安装与运行

E_INK_Timer 采用**单文件架构**，无需安装任何依赖或服务器环境。

- **桌面端**：直接使用现代浏览器（推荐 Chrome, Edge, Safari）打开 `.html` 文件即可。
- **移动端**：可将 HTML 文件发送至手机，使用浏览器打开。以Safari为例，打开网址https://0963hk.github.io/E-INK-Timer/，右下角三个点-共享-添加到主屏幕。

### 1.2 界面概览

- **中央区域**：核心计时器、任务输入框、控制按钮。
- **左侧侧边栏**：辅助组件（天气、下班倒计时、活动日志、自动存档设置）。
- **顶部栏**：全局设置（主题切换、禅模式）。
- **底部区域**：每日数据统计与可视化图表。

## 2. 核心功能

### 2.1 计时模式

系统支持两种计时逻辑，适用于不同的工作场景：

- **倒计时** ：
    - 适用于已知时长的任务（如番茄工作法）。
    - 在 `TIME` 输入框设置分钟数（默认 25 分钟），点击 `START` 开始。
- **正计时**：
    - 适用于时长不确定的任务。
    - 勾选 `COUNT UP` 复选框，时间将从 `00:00` 开始累加。
    - 点击 `DONE` 结束任务并记录实际耗时。

### 2.2 任务管理

- **输入任务**：在 `TASK` 输入框填写当前专注内容（如 "Coding", "Reading"）。
- **提前完成**：若在倒计时结束前完成了任务，**请务必点击 `DONE` 按钮**（而非 RESET）。系统将记录**实际消耗的时间**而非预设时间，确保统计数据的真实性。
- **重置 (Reset)**：点击 `RESET` 将强制停止计时、归零时间并恢复默认设置，此次操作**不会**被记录到日志中。

### 2.3 禅模式

- 点击顶部 `[ZEN]` 按钮或直接点击巨大的时间数字。
- **效果**：隐藏除时间显示外的所有 UI 元素，提供极致的沉浸体验。
- **退出**：再次点击屏幕或按下 `Enter` (回车键)。

## 3. 组件与工具

> 注意：在移动端或窄屏设备上，左侧组件栏默认折叠。点击左上角的 [ - ] WIDGETS 按钮可展开/收起。
> 

### 3.1 每日活动日志 (Today's Log)

- **查看**：实时显示今日所有专注记录（按时间倒序）。
- **编辑 [E]**：点击记录旁的 `[E]` 可修正任务名称或时长。修改后，底部的统计图表会自动同步更新。
- **删除 [X]**：点击 `[X]` 可永久删除该条错误记录。
- **手动保存**：点击标题栏的 `[SAVE]` 可立即导出今日日志为 `.txt` 文件。

### 3.2 下班倒计时 (Day Left)

- **设置**：在中央输入区右侧设定您的 `END OF DAY` 时间（默认 18:00）。
- **显示**：左侧组件将实时显示距离下班的剩余时长。若少于 1 小时，将精确显示到秒。

### 3.3 实时天气 (Weather)

- **激活**：首次点击 `WEATHER` 组件区域。
- **权限**：浏览器将请求地理位置权限。允许后，系统将通过 Open-Meteo API 获取您当前坐标的实时天气（不收集隐私数据）。
- **刷新**：再次点击组件即可刷新数据。

## 4. 数据与统计 (Data & Stats)

### 4.1 每日统计图

- 底部区域会自动生成今日专注时间的饼状图 。
- **分享**：点击 `[SHARE IMG]` 按钮，系统将自动生成包含图表、总时长和日期的 PNG 图片并下载，方便分享至社交媒体。

### 4.2 自动归档系统

系统支持每日定时自动导出数据，实现“无感结转”。

- **设置**：在左侧日志栏底部找到 `AUTO:` 设置项，设定触发时间（推荐默认 `00:00`）。
- **运行逻辑**：
    - 若设为 `00:00`：系统会在午夜自动导出**昨天**的日志(.txt)和统计图(.png)，并清空界面迎接新的一天。
    - 若设为 `23:59`：系统会在当天结束前自动保存当日数据。
- **重要提示**：浏览器可能会拦截自动下载。首次触发时，请在浏览器提示框中选择“允许下载多个文件”。

## 5. 数据隐私声明

- **Local-First**：E_INK_Timer 不包含任何后端服务器代码。
- **本地存储**：所有数据（日志、设置、主题偏好）均存储在您浏览器的 `LocalStorage` 中。清除浏览器缓存可能会导致历史数据丢失，请定期使用导出功能备份。
